## Creating a New ADR

1. Use the standard ADR template formats defined in the `architect-brainstorm` skill.
2. Name the file `NNNN-short-title.md`.
3. Fill in the Context, Options, Decision, Rationale, and Consequences.
4. Update this index table after creation.

## ADR Status

- **Proposed**: Under discussion
- **Accepted**: Decision made, implementing
- **Deprecated**: No longer relevant
- **Superseded**: Replaced by another ADR
- **Rejected**: Considered but not adopted
